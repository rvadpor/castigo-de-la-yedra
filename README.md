\# Castigo de la Yedra



Videojuego tipo escape room desarrollado de forma individual utilizando Unity y C#.



\---



\## Descripción



Castigo de la Yedra es un proyecto centrado en la resolución de acertijos y exploración, donde el jugador debe interactuar con distintos elementos del entorno para avanzar.



El proyecto ha sido desarrollado como práctica de programación, arquitectura de sistemas y gestión de estados dentro de Unity.



\---



\## Tecnologías utilizadas



\- Unity

\- C#

\- MySQL

\- Git

\- Programación orientada a objetos



\---



\## Características



\- Sistema de acertijos

\- Gestión de interacción

\- Persistencia de datos

\- Arquitectura modular

\- Gestión de estados de partida

\- Desarrollo individual completo
